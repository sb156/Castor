# Castor Repository

A collection of hardened, high-performance userscripts designed for Tampermonkey

## Current Tools

### 1. YT-NAV (YouTube Video Navigator)
Shortcut mapping and simple GUI for video timestamps.

* **[0]**: Reset to start (0:00).
* **[1]**: Jump back 2 minutes.
* **[2]**: Jump forward 2 minutes.
* **[.]**: Jump forward 100 minutes (Skip to end).

### 2. YouTube Chat Optimizer
Strips high-overhead (avatars/badges) and throttles rendering to maintain system stability during high-traffic events.
(Makes Youtube chat less glitchy)

Installation
1. Copy the raw `.user.js` code from this repository.
2. Create a new script on Tampermonkey dashboard and paste the code.
3. Save and refresh the target page.
